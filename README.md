Rule-Based Python Chatbot 🤖


A lightweight, dictionary-based AI assistant built with Python. This chatbot uses simple rule-based logic to interact with users, offer greetings based on the time of day, and provide answers to common programming and general life questions.

🌟 Features
    --Time-Aware Greetings: Automatically detects the time of day to greet you with "Good Morning," "Good Afternoon," etc.

    --Knowledge Base: Pre-programmed with responses for:

    --General Chit-chat: Greetings, jokes, riddles, and facts.

    --Emotional Support: Responses for when you feel sad, stressed, or happy.

    --Python Concepts: Quick definitions for variables, loops, dictionaries, and more.

    --Keyword Matching: Uses a flexible search to find answers even if your sentence is long.

    --Simple Loop: Runs continuously until you say "Bye." 
    
A simple rule-based Python chatbot that responds to everyday conversations, motivational prompts, and basic programming questions.
